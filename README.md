# Analise Racial Pro - Negram
A plataforma Análise Racial Pro, do Projeto Negram, é um WebGIS interativo para visualização de dados sociodemográficos. Executada no navegador, permite filtrar municípios, analisar camadas raciais. Este sistema transforma arquivos locais em mapas de calor e métricas precisas para apoiar as pesquisas e analises do Negram.
